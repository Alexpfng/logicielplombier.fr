

# logiciel-plombier.fr — Site vitrine premium pour Bulbiz

## Vue d'ensemble
Site éditorial de conversion ciblant les plombiers et chauffagistes, avec un design bleu profond/blanc, mobile-first, rassurant et orienté terrain. Le site positionne Bulbiz comme LA solution de centralisation des demandes clients.

## Design System
- **Palette** : Bleu royal (#2563EB) pour les CTA, bleu marine (#1E3A8A) pour les headers, fond gris-bleuté (#F8FAFC), texte ardoise (#0F172A)
- **Typo** : Inter — titres bold avec letter-spacing négatif, corps lisible line-height 1.6
- **Composants** : Cartes avec ombres légères, coins arrondis 16px, beaucoup d'air, micro-animations subtiles
- **Mobile-first** : Sections empilées, CTA toujours accessibles, typographie responsive avec clamp()
